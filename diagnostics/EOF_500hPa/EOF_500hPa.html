EOF of geopotential height anomalies of 500 hPa

This computes the climatological anomalies of 500 hPa geopotential height, then calculates the EOFs using NCL's eofunc.
The code is in NCL and requires model input: 1) monthly averaged surface pressure (ps), 2) monthly averaged geopotential height (zg)
