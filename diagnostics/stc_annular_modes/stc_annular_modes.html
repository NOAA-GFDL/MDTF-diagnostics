Seasonality and extremes of vertically propagating planetary waves

This POD computes annular mode indices and diagnostics relating to coupling with the stratosphere. The annular modes are assumed to be the first EOF of daily zonal mean geopotential height anomalies. Please see the POD documentation for how anomalies are computed.

EOF1 Vertical Structure

These plots show the latitudinal structure of the 1st EOF of the zonal mean geopotential height anomalies. The latitude "profiles" are only plotted for a limited set of standard pressure levels to aid comparison between model and reanalysis datasets. They are intended to be qualitative depictions of the annular mode structures; the vertical scale of the plots and magnitude of displacements of the profiles are not necessarily meaningful.

< color=navy>Annular Mode Predictability

These plots show the predictability of the annular mode at a tropospheric pressure level (by default 850 hPa, but this value is configurable in the POD settings.jsonc file). They depict the fraction of the variance of the 10-40 days ahead mean annular mode index for the given pressure level that can be "predicted" from a persistence forecast using the instantaneous annular mode index at other levels.
